### 배열 (Array) ###
----------
배열 (Array)란 동일한 특성을 가지며 일정한 규칙에 따라 몇몇 요소가 나열되어 있는 `데이터 집합`입니다. 배열을 선언함으로써 `다수의 데이터`를 처리할 수 있습니다. C#에서 배열은 다음과 같이 선언할 수 있습니다.
<br><br>
`데이터형식[ ] 배열이름 = new 데이터형식[ 용량 ];`

{% highlight c# %}
int[] array = new int[5];
{% endhighlight %}

{% highlight c# %}
int[] array = new int[5] {1, 2, 3, 4, 5};
{% endhighlight %}

{% highlight c# %}
int[] array = new int[5];
array[0] = 1;
array[1] = 2;
array[2] = 3;
array[3] = 4;
array[4] = 5;

{% endhighlight %}
