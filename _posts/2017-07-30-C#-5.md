#### 프로젝트 구성 ####
[![1]({{ site.images }}/C/ch5/1.png)]({{ site.images }}/C/ch5/1.png)
도구상자에서 **Group Box, RadioButton, CheckBox, Button**을 **Form1**에 생성합니다. 위의 이미지와 같이 배치합니다.



[![2]({{ site.images }}/C/ch5/2.png)]({{ site.images }}/C/ch5/2.png)
속성을 다음과 같이 설정합니다.
1. GroupBox (1번)

	* Text : 결제방식
	
	
2. RadioButton (1번)

	* Text : 현금
	
	
3. RadioButton (2번)

	* Text : 카드
	
	
4. RadioButton (3번)

	* Text : 쿠폰
	
	
5. GroupBox (1번)

	* Text : 현금
	
	
6. CheckBox (1번)

	* Text : 짜장
	
	
7. CheckBox (2번)

	* Text : 짬뽕
	
	
8. CheckBox (3번)

	* Text : 탕수육	
	

9. RadioButton (4번)

	* Text : 상태
	* AutoCheck : False
	

10. Check (4번)

	* Text : 활성화

		
11. Button

	* Text : 제출
	
	
	
- Tip : RadioButton의 AutoCheck는 클릭으로 단추의 상태가 변경되지 않습니다.
